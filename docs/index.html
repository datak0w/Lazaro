Manual de Lazaro AI

Guía completa para usar el asistente de voz libre. Pensada para leerse con lector de pantalla (NVDA, JAWS, TalkBack, VoiceOver) o con teclado.

Importante Lo que importa para tu seguridad al caminar (pitidos, cámara, GPS, rutas) corre en el teléfono, sin esperar a un servidor. La IA solo ayuda en el diálogo y la memoria.

Cómo leer este manual con lector de pantalla

Este sitio usa encabezados, listas y regiones (landmarks) para que puedas saltar rápido. Detalle en Accesibilidad de este manual.

  • NVDA / JAWS (Windows): tecla H para recorrer títulos; D para regiones; K para enlaces.
  • TalkBack (Android): explorar por encabezados o por enlaces desde el menú de lectura.
  • VoiceOver (iPhone/Mac): rotor → Encabezados o Enlaces.
