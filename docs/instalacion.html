Instalación

Lazaro AI es una app Android. Se instala desde APK (aún no está en Play Store).

Requisitos

  • Android 8 (API 26) o superior
  • Micrófono y altavoz (o auriculares)
  • Google Maps instalado (para navegación urbana)
  • Opcional: Pixel 9 para profundidad ARCore; Samsung A34 sirve para desarrollo monocular

Instalar el APK

  1. Consigue el APK (compílalo o pídelo a quien mantiene el proyecto).
  2. En el móvil, permite instalar apps de orígenes desconocidos para el explorador o gestor de archivos que uses.
  3. Abre el APK e instálalo.
  4. Abre Lazaro AI → Iniciar asistente.
  5. Concede los permisos que pida.

Instalar desde el PC (desarrolladores)

Comandos en una terminal:

git clone https://github.com/datak0w/Lazaro.git
cd Lazaro
./gradlew assembleDebug
adb install -r app/build/outputs/apk/debug/app-debug.apk
Nota La clave de Gemini (opcional) se configura en local.properties al compilar. Ver Configuración.

Lista de comprobación tras instalar

  • Iniciar asistente y probar «Lazaro, qué hora es»
  • Activar acceso a notificaciones (Maps + WhatsApp)
  • Probar «dónde estoy» con GPS al aire libre
  • Instalar Google Maps si quieres giros hablados
