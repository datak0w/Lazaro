Accesibilidad de este manual

Este manual está pensado para personas ciegas o con baja visión que usan lector de pantalla o solo teclado.

Qué hemos hecho para que sea usable

  • Idioma en español declarado en cada página (lang="es").
  • Un solo título principal (h1) por página; secciones con h2 y h3 ordenados.
  • Regiones: cabecera, menú de navegación, contenido principal y pie.
  • Menú en lista de enlaces (no solo texto suelto), para navegar por lista o por enlace.
  • Enlaces de salto al inicio de cada página (contenido, menú, pie).
  • Miga de pan para saber dónde estás.
  • Tablas con título (caption) y encabezados de columna.
  • Avisos con etiqueta hablada («Importante», «Atención», «Consejo»), no solo color.
  • Enlaces con texto claro (evitamos «pincha aquí»).
  • Foco visible al tabular (resaltado amarillo/negro).
  • Contraste alto y respeto a «reducir movimiento» del sistema.

Atajos habituales con lector de pantalla

Navegación rápida por tipo de elemento
Lector Títulos Regiones / landmarks Enlaces
NVDA H (Shift+H atrás) D K
JAWS H R (regiones) Insert+F7 lista de enlaces
TalkBack Menú de lectura → Encabezados Menú de lectura → Controles Menú de lectura → Enlaces
VoiceOver Rotor → Encabezados Rotor → Contenedores / landmarks Rotor → Enlaces

Orden recomendado de lectura

  1. Enlace «Saltar al contenido principal» (si usas teclado).
  2. Título h1 de la página.
  3. Recorrer h2 de esa página.
  4. Si quieres otra sección: menú «Secciones del manual» o volver a Inicio.

Imágenes

Las figuras del manual tienen texto alternativo descriptivo. Si una imagen solo decora, se marca como decorativa (alternativa vacía) para que el lector la ignore.

PDF del checklist GoPro

El checklist de captura GoPro (PDF) es un documento aparte. Los lectores de pantalla suelen leer PDF etiquetado; si encuentras problemas, abre también la versión HTML checklist en HTML.

Informar de un problema de accesibilidad

Si algo no se lee bien o falta un encabezado, abre una incidencia: GitHub Issues de Lazaro y describe el lector (NVDA, TalkBack…) y la página.

Consejo La app Lazaro AI está pensada para usarse por voz, sin mirar la pantalla. Este manual es para aprender o consultar desde el PC o el móvil con lector.
